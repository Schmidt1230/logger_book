# Backend
LoggerBook uses NodeJS for backend. And also MongoDB for cloud data storage.

## NodeJS dependencies:

<pre>
<code>npm install express mongoose body-parser</code>
</pre>


<pre>
<code>npm install bcrypt jsonwebtoken</code>
</pre>

<pre>
<code>npm install nodemon</code>
</pre>
